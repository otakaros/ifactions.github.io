Introduction

We introduce you to the iFactions Network
This presentation is only informative to familiarize you with our systems,
it does not serve as a Wikipedia.

The server is inspired by HCF servers, so that even beginners can handle the server

Server style: Semi-hcf
Kit maps: Protection 1 / Sharpness 1, No Debuff
Map size: 2000 x 2000
End portals: 750, -750 & -750, 750
Koths: -200, -700 & 200, 700

TIP: If you have never played Fractional/HCF servers,
we recommend checking out our video tutorial.

Classes

- There are 4 different classes on the server (archer, bard, rogue, miner)
- Each class has its own specific features and abilities
- To obtain these classes you need to equip a certain armor
- Once you have the armor, you will receive the class

> Specific features of these classes can be found here <

Koths

- These are faction events

- The strongest faction to capture the Koth will receive the Koth Loot

- Koths run automatically every day at 18:00

- You can find out the Koth Loot using the /koth loot [koth_name] command.

Heal potions

- Simplified crafting of Heal potions

- Only 1 gold nugget is needed for crafting ingredients

- Faster potions

- Making PvP easier

StatTrak system

- You can look forward to StatTrak system

- All kills are counted and recorded on your sword

Combat system

- No one on the server will disconnect you in a combat or escape to the Safe Zone (red wall will appear)

If a player disconnects in Combat, they die

- Combat Tag: 30 seconds

- Ender Pearl: 15 seconds

DTR system

- You can conquer enemy factions on the server using the DTR system
- Each faction has its own DTR, which depends on the number of players in the faction
- The maximum achievable DTR is 5.50

- For every death, 1 DTR is deducted and the DTR regen is Frozen for 30 minutes
- If the DTR drops below 0, the faction automatically becomes Raidable
and you can select all enemy territory

- Regen DTR = 3 per hour (up to 6 per hour with higher member counts)

TIP: Make sure the members in the faction don't die unnecessarily, so you can easily become raidable.

Custom End

- On the server you can find custom end

- In the end you can farm ender pearls for pvp,
gunpowder for brewing potions and slimeballs for sticky pistons

- The end is accessed through 2 end portals from the main world

Dynmap

- On the server you will find a dynmap,
You can use the dynmap to locate enemy territories

- You can register on the dynmap to use the online chat,
if you are on your mobile phone away from home, for example.

Spawner wrench

- You can pick up a spawner on the server, using the Spawner Wrenche

Gambling

- You can gamble on the server using the Gambling Key,
which you get for emeralds.

Daily rewards

- On the server you will find daily rewards,
which you can collect using /dailybonus

- The rewards are escalating every day,
each reward collected in a row is all the more precious.

Voting

- You can vote on the server, for each vote you will receive a reward

- The rewards increase with the number of votes,
for a certain number of votes you will receive more and more rare rewards

- The player with the most votes at the end of the month will receive a premium reward

Harvester

- The server is simplified harvesting
- You activate the function using /harvester

Looting XP

- On server looting adds not only more loot but also experience

- You get more XP by looting, each level adds more XP

Mob stacking

- On the server you will find stacking mobs from spawners

Sword system

- You can rename your sword to a random nickname
by right-clicking with your sword on the beacon

Hall of Fame

- The server has a Hall of Fame

- Players with the most kills are posted here

Other systems

- Fast login for players with Original Minecraft
using the /fastlogin command

- Automatic Nether portal cancellation,
if you teleport into a built-up portal.
(just stay in the portal and it will shut itself down)

- Disabled glitching with ender pearls

Support - Ticket system

- On our community discord server you can find the #ticket_support room,
where you can create a room that only you and the Admin Team will have access to

- In case of any "serious" issue, please contact us there

- Tickets are for serious issues only!
Nonsensical tickets are therefore deleted
